# PixelResizer
無劣化でピクセル画像を拡大するソフトです。

.NETランタイムが同梱されたバージョンと同梱されていないバージョンがあります。
同梱されいないバージョンを利用する場合はリリースに書かれた.NETランタイムを導入してください。

## ライセンス
MIT ライセンス

著作権 (C) BreezySoda

## システム要件
* x86またはx64のWindows 7以降のシステム
* 最低200KBのROMスペース(.NETランタイムが未同梱の場合)
* 最低500MBのRAMスペース

## 使用方法
* **画像を読み込む**をクリックして処理する画像を選択する
* **倍率**を1から100までの範囲で変更する
* **画像を保存する**をクリックして処理後の画像を保存する
