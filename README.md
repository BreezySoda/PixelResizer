# PixelResizer
無劣化でピクセル画像を拡大するソフトです。
自分でビルドする際は **Framework** ではない方の **.NET** を使用してください。

検証していませんが、ピクセル画像以外でも使用できる可能性があります。

## システム要件
* ビルド時の.NETが導入されたWindowsシステム
* 100MBから500MBほどのRAMスペース

## サンプル画像
**左が処理前(32x32)、右が処理後(256x256)**

## 使用方法
* **画像を読み込む**をクリックして処理する画像を選択する
* **倍率**を1から100までの範囲で変更する
* **画像を保存する**をクリックして処理後の画像を保存する
