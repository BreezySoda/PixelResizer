# PixelResizer
無劣化でピクセル画像を拡大するソフトです。
**.NET**を使用してください。

開発者はファイルを1つにまとめるのを諦めました。
## ライセンス
MIT ライセンス

## システム要件
* 最低500MBのRAMスペース

## 使用方法
* **画像を読み込む**をクリックして処理する画像を選択する
* **倍率**を1から100までの範囲で変更する
* **画像を保存する**をクリックして処理後の画像を保存する
